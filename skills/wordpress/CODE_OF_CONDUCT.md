# Code Of Conduct

## Our Standard

This project should be a practical and respectful place to collaborate.

Please:

- Be clear, kind, and constructive.
- Assume good intent while still reviewing carefully.
- Focus criticism on code, docs, behavior, and outcomes.
- Respect differences in experience, background, tools, and workflow.

Please do not:

- Harass, threaten, insult, or demean others.
- Post private information without permission.
- Derail technical discussions with personal attacks.
- Use sexualized, discriminatory, or hostile language.

## Enforcement

Maintainers may edit, hide, or remove comments, issues, pull requests, or other contributions that violate this code of conduct. Repeated or severe violations may lead to blocking from the project.

## Reporting

If you see behavior that violates this code, contact a maintainer through the least public channel available for the repository. If no private channel exists, open a brief issue without sensitive personal details.
